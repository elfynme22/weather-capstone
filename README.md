# Weather Capstone (Express + Axios + EJS)

A simple website that checks **tomorrow’s rain probability** for a given city using the **Open-Meteo API**.

## Features

- Search by city name
- Shows tomorrow’s rain probability + min/max temperature
- Displays a 7-day forecast in cards
- Handles errors (invalid city, missing data, API issues)

## Tech Stack

- Node.js
- Express.js
- Axios
- EJS
- Nodemon (dev)

## How to Run

Install dependencies:

```bash
npm i
```
